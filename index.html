📖 इस्तेमाल कैसे करें (Hindi Usage Guide)

  1. Subscription जोड़ें: ऊपर form में नाम, category, cost, billing cycle, next renewal date और last used date भरें, फिर "Add Subscription" दबाएं।
  2. Usage Frequency ज़रूर चुनें: यह बताता है कि आप उस service को कितना इस्तेमाल करते हैं — इसी से waste score calculate होता है।
  3. Waste Status समझें: हर card पर रंगीन badge दिखेगा — नीचे legend देखें।
  4. Dashboard देखें: ऊपर आपको Monthly Spend, Yearly Spend, Wasted Amount और Potential Savings दिखेंगे।
  5. AI Insights: "Get AI Insights" बटन दबाकर personalized सलाह पाएं — API key न हो तो भी free analysis मिलेगा।
  6. Data सुरक्षित रखें: "Export JSON" से backup लें। सारा data आपके browser (LocalStorage) में ही save होता है, कहीं और नहीं भेजा जाता।
  7. Renewal Reminder: अगर किसी subscription का renewal 3 दिन में आने वाला है, तो ऊपर banner में alert दिखेगा।
